# Click To Change Theme
 # 🎨 Color Background Theme Switcher

A simple and responsive background theme switcher. Click on any color box to instantly change the background of the page. Clean UI and fully responsive design built using HTML, CSS, and JavaScript.

## 🚀 Features

- Click to change background color
- Responsive layout for all screen sizes
- Clean and minimal design
- Easy to customize and extend

## 📂 Files

- `index.html` – Markup structure
- `style.css` – Styling and responsiveness
- `script.js` – Theme switching logic
